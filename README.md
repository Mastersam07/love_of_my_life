A dart package to add the love of your life to your dart projects.

## 🎖 Installing

```cmd
flutter pub add love_of_my_life
```

#### OR

```yaml
dependencies:
  love_of_my_life: "^0.0.1"
```

### ⚡️ Import

```dart
import 'package:love_of_my_life/love_of_my_life.dart';
```
